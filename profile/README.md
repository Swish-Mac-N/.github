## 📋 Overview

<a href="#swish">Swish</a> is a window management application that uses trackpad and mouse gestures rather than keyboard shortcuts or manual dragging to position, resize, and arrange windows on Mac — providing an intuitive, physical interaction model for the window management task that most Mac users perform constantly throughout their day.

The <a href="#swish">gesture-based window control</a> is Swish's defining capability — drawing simple gestures on the trackpad with a window's title bar selected positions that window into specific screen locations and sizes. A swipe left snaps a window to the left half of the screen, a swipe to a corner positions it as a quarter-screen window, and various gesture patterns map to the full range of window arrangements that manual dragging and resizing would otherwise require multiple separate actions to achieve. This gesture vocabulary, once learned, becomes significantly faster than the click-drag-resize sequence that manual window management requires, particularly for users managing multiple windows simultaneously throughout their workday. The <a href="#swish">screen snapping and quarter-tiling</a> provides precise window positioning to half-screen, quarter-screen, and other standard arrangements — the layouts most useful for side-by-side window comparison, multitasking, and organized desktop arrangement, achieved through a single gesture rather than manual measurement and positioning.

The <a href="#swish">window stacking and cycling</a> manages multiple windows occupying the same screen position — stacking windows in a layered arrangement and cycling through them with additional gestures, useful for managing more windows than screen space comfortably accommodates simultaneously. The <a href="#swish">multi-monitor support</a> extends gesture-based window management across multiple connected displays — moving windows between monitors and positioning them within each display's available space using the same gesture vocabulary that works on a single screen.</p>

<a href="#swish">Customizable gesture mapping</a> allows configuring which specific gestures trigger which window arrangements — adapting Swish's gesture vocabulary to personal preference or muscle memory from other tools, rather than requiring adaptation to a fixed gesture set. <a href="#swish">Mouse gesture alternative</a> provides gesture-based window management for mouse users without a trackpad, extending the gesture interaction model beyond trackpad-equipped MacBooks to desktop Mac setups with traditional mice.

<a href="#swish">Keyboard shortcut integration</a> complements the gesture system with traditional keyboard shortcuts for window management actions, providing flexibility for situations where gesture input isn't convenient or for users who prefer keyboard-driven workflows for specific actions. <a href="#swish">Visual gesture feedback</a> displays the target window position and size during gesture input, providing immediate visual confirmation of where a window will land before completing the gesture.</p>

## 🤔 FAQ

<details>
<summary>How does Swish position windows?</summary>

Swish uses trackpad or mouse gestures drawn while a window's title bar is selected to position and resize windows into specific screen arrangements.

</details>

<details>
<summary>Does Swish work with a regular mouse, not just a trackpad?</summary>

Yes. Mouse gesture support extends Swish's gesture-based window management to desktop Mac setups without a trackpad.

</details>

<details>
<summary>Can Swish manage windows across multiple monitors?</summary>

Yes. Multi-monitor support allows moving and positioning windows across multiple connected displays using the same gesture vocabulary.

</details>

<details>
<summary>Are Swish's gestures customizable?</summary>

Yes. Gesture mapping can be configured to assign specific gestures to specific window arrangements based on personal preference.

</details>

<details>
<summary>Does Swish support window stacking?</summary>

Yes. Window stacking and cycling manages multiple windows occupying the same screen position with gesture-based cycling between them.

</details>

<details>
<summary>Is Swish available through Setapp?</summary>

Yes. Swish is available both through the Setapp subscription bundle and as a standalone purchase.

</details>

<details>
<summary>Does Swish replace keyboard shortcuts for window management?</summary>

No. Swish complements gesture-based control with traditional keyboard shortcuts, providing flexibility for different situations.

</details>

<details>
<summary>How long does it take to learn Swish's gestures?</summary>

Learning curve varies by user, but most users report the gestures becoming automatic muscle memory within a relatively short period of regular use.

</details>

<details>
<summary>Does Swish show feedback during gesture input?</summary>

Yes. Visual gesture feedback displays the target window position and size during the gesture before it completes.

</details>
